批量粘贴队列（1.3）

三种模式可在顶栏或全局快捷键间切换；托盘图标与列表高亮会随模式变色，便于扫一眼确认状态。

模式说明

普通 多选后 Enter 按列表顺序连续粘贴；可与「合并相邻纯文本」等选项配合。
FIFO 多选 Enter 入队。在任意目标窗口中，每次 Ctrl+V 或 Shift+Insert 出队一条并更新剪贴板。列表显示序号角标；新复制也可按规则自动入队。
LIFO 入队方式与快捷键相同，出队顺序为后进先出。

默认全局循环切换键为 Alt+/（可在设置中修改）；顺序为 普通 → LIFO → FIFO → 普通，与面板顶栏一致。

模式主色（顶栏标签、角标、托盘、面板内列表选中/悬停混合）：普通 #139493 · FIFO #2563EB · LIFO #CA8A04。托盘在 FIFO/LIFO 下会叠加 F / L 标记。

功能

按场景拆开；与 Ditto、CopyQ、Listary 等差异见下文表格。

剪贴板

不抢焦点

扩展样式与消息处理配合，使弹窗在多数操作中不激活为前台窗口，减轻 IDE 与输入法被打断的情况。

靠近插入点

优先根据插入符位置摆放面板，失败时再退回鼠标位置或配置项。

过滤

弹窗出现后可直接键入过滤；中文支持全拼与首字母索引。

存储

文本、图片（缩略图）、路径列表等；SQLite 持久化；默认可存数千条并去重置顶。

主题

暗色界面接近 VS Code Dark+ 的底与对比度；强调色与上述模式色一致。亮色为浅灰底。可跟随系统。

快捷短语

将常用条目固定为短语并单独视图管理；侧栏检索同样避免抢焦点。

文件对话框跳转

Shell 导航

在支持的公共对话框中通过原生 DLL 与 Shell 接口直接改目录，减少对地址栏模拟的依赖。

多款管理器

对常见双栏与增强资源管理器维护专用或半专用路径采集；其余按白名单遍历。

WPS

对非标准对话框走多条 UI Automation 与控件回退路径。

自动化选项

可选：对话框首次前台时自动拉路径；框内第一次点击自动跳转；从资源管理器切回时同步目录等。

兼容模式

可关闭注入，仅保留键入/地址栏模拟，便于在严格安全软件环境中使用。

收藏与历史

收藏目录与最近路径进入同一跳转列表，支持检索。
